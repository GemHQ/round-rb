## Installing round-rb:


### Prerequisites:

* Ruby 2.1.5

### Linux (debian-based, tested on Ubuntu 14.04)

1. Install system dependencies (*this is the only time you need sudo!*)

  ```bash
  $ sudo apt-get install gcc make 
  ```

2. Install the client

  ```bash
  $ gem install round
  ```

 [[back]](../README.md)

### Mac OSX:

1. Gem install the client & rejoice!
  ```bash
  $ gem install round
  ```
